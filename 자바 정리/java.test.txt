썬 마이크로시스템즈의 제임스 고슬링과 다른 연구원드이 개발한 객체지향적 프로그래밍 언어
자바 가상 머신 jvm (java Virtual Machine, JVM)
자바를 다른 컴파일언어와 구분 짓는 가장 큰 특징은 컴파일된 코드가 플랫폼 독립적
모든 자바 프로그램은 cpu나 운영 체제의 종류와 무관하게 동일하게 동작할 것을 보장한다
현재 웹 애플리케이션 개발에 가장 많이 가숑하는 언어 가운데 하나
모바일 기기용 소프트웨어 개발에도 널리사용


유닉스 기반의 배경을 가지고 있었기 때문에 문법적인 특성은 파스칼이 아닌 C++의 조상인 C 언어와 비슷하다
. 자바 컴파일러는 자바 언어로 작성된 프로그램을 바이트코드라는 특수한 바이너리 형태로 변환한다. 바이트코드를 실행하기 위해서는
JVM(자바 가상 머신, JAVA Virtual Machine) 이라는 특수한 가상 머신이 필요한데, 이 가상 머신은 자바 바이트코드를 어느 플랫폼에서나 동일한 형태로 실행
